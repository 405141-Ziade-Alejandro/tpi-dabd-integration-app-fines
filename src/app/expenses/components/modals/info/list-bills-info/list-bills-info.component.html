Información del Listado de Gastos

¿Qué es este listado?

Este listado te permite gestionar y monitorear los gastos del sistema, diferenciando entre gastos asociados a empleados y proveedores de servicios. Cada registro contiene información detallada y permite diferentes acciones según su estado.

Columnas del Listado
  • Tipo: Clasificación entre gastos Ordinarios y Extraordinarios.
  • Proveedor: Puede ser un Empleado o un Proveedor de servicios.
  • Monto: Valor monetario del gasto en pesos argentinos ($).
  • Período: Mes y año correspondiente al gasto.
  • Categoría: Clasificación específica para análisis contable.
  • Fecha: Día de registro del gasto.
  • Estado: Puede ser Nuevo, Cancelado o Activo.
Acciones por Registro
  • Ver Detalle: Disponible para todos los registros.
  • Editar: Permite modificar la información en cualquier estado.
  • Eliminar: Solo disponible para registros en estado Activo.
  • Activar: Solo disponible para registros en estado Nuevo.
Funcionalidades Adicionales
  • Exportación: - Botón verde para formato Excel - Botón rojo para formato PDF
  • Filtros: Búsqueda disponible por todas las columnas y tipo de proveedor.
Nota:

Las acciones disponibles para cada registro varían según su estado. Utilice los filtros superiores para facilitar la búsqueda de registros específicos.
